Middleware Chain (applied to every /api/* call)

  1. errorHandler — catches exceptions, returns 500 JSON
  2. requestLogger — FNV-1a IP hash, latency, status → ApiAudit store
  3. authGate — write methods require x-api-key (constant-time compare)
  4. jsonBody — parses JSON request body, sets Content-Type
  5. corsHandler — injects Access-Control-Allow-* headers
  6. dispatcher — routes /api/* to handler functions
